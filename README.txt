file generation rule:
sha256(seq) + 0

## auto_store.sh
* file_size - each file's size to be generated. unit: bytes.
* total - number of files to upload
* start_seq - from which sequence number to start with.
* lotus_output - where raw file is generated under.
* miner_id - specify the dedicated miner to accept.
* lotus_user - lotus daemon server user
* lotus_host - lotus daemon server host
